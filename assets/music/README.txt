Folder ini untuk file musik background.

## Cara Menambahkan Musik:

1. Siapkan file musik format MP3
2. Rename file menjadi: **background.mp3**
3. Copy file ke folder ini
4. Music player di website akan otomatis berfungsi

## Rekomendasi:

- Format: MP3
- Bitrate: 128kbps (untuk ukuran file lebih kecil)
- Durasi: 3-5 menit (akan loop otomatis)
- Jenis musik: Instrumental, lagu rohani, atau musik klasik yang cocok untuk acara syukuran

## Sumber Musik Gratis:

- YouTube Audio Library
- Free Music Archive
- Incompetech
- Bensound

Pastikan musik yang digunakan bebas copyright atau Anda memiliki izin penggunaan.
